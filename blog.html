Question-1: What are the different ways to select an element in the DOM?

In JavaScript, there are multiple ways to select an element in the DOM. Some of the most common ways are:
1. getElementById(): This method is used to select an element by its id attribute.
2. getElementsByClassName(): This method is used to select elements by their class name.
3. getElementsByTagName(): This method is used to select elements by their tag name.
4. querySelector(): This method is used to select the first element that matches a specified CSS selector.
5. querySelectorAll(): This method is used to select all elements that match a specified CSS selector.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The innerHTML property sets or returns the HTML content inside an element. The innerText property sets or returns the text content that is only visible text inside an element. The textContent property sets or returns the all text content,including the hidden elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a mechanism in which an event is first captured and handled by the innermost element and then propagated to outer elements. It is the default behavior of the browser.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element, you can use the document.createElement() method. To add an element to the DOM, you can use the appendChild() method . To remove an element from the DOM, you can use the removeChild() method.
